# React Bebidas

Este proyecto es una práctica de React para aprender y reforzar conceptos fundamentales del framework. La aplicación permite buscar recetas de bebidas y ver sus detalles utilizando componentes, contextos y hooks de React.

## Características
- Búsqueda de recetas de bebidas por nombre o categoría
- Visualización de detalles de cada receta
- Uso de Context API para manejo de estado global
- Componentes reutilizables y organizados
- Estilos personalizados con CSS y Bootstrap

## Estructura del proyecto
```
src/
	App.jsx
	components/
		Formulario.jsx
		Header.jsx
		ListaRecetas.jsx
		Receta.jsx
	context/
		CategoriaContext.jsx
		ModalContext.jsx
		RecetasContext.jsx
	assets/
	App.css
	bootstrap.min.css
	index.css
	main.jsx
```

## Instalación y uso
1. Clona el repositorio o descarga el código.
2. Instala las dependencias:
	 ```bash
	 npm install
	 ```
3. Inicia el servidor de desarrollo:
	 ```bash
	 npm run dev
	 ```
4. Abre tu navegador en `http://localhost:5173` (o el puerto indicado en consola).

## Tecnologías utilizadas
- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [Bootstrap](https://getbootstrap.com/)
